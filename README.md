# 晏阳城市建设 官方网站

晏阳城市建设服务器的官方宣传网站，基于 Minecraft Fabric 1.20.1，专注城市规划与轨道交通创作。

## 技术栈

- React 19
- React Router DOM 7
- Vite 7
- 纯 CSS（无框架依赖）

## 本地开发

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

构建产物输出至 `dist/` 目录。

## 页面结构

- **首页** - 服务器介绍、核心特色、数据统计
- **活动** - 服务器历史动态与版本更新时间线
- **加入** - 入服流程、服务器配置信息、常见问题
- **关于** - 社群介绍、轨道交通成就、合作伙伴展示
